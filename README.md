# 文件管理系统前端项目
<img width="1452" height="771" alt="屏幕截图 2025-11-25 142355" src="https://github.com/user-attachments/assets/5ac30185-91cc-4d83-a68c-0475825b969c" />
一个基于 Vue3 + Vite 的致力于打造安全高效的个人文件管理系统，支持文件/文件夹的查看、创建、删除、重命名、移动、上传、下载等各项操作。

## ✨功能特点
- 直观的文件列表展示（区分文件和文件夹）
- 面包屑导航，支持快速跳转目录
- 完整的文件操作：新建文件夹、上传文件、下载文件、查看文本内容等
- 登录状态管理与路由守卫

## 👩‍💻技术栈
- Vue 3（Composition API）
- Element Plus（UI 组件库）
- Pinia（状态管理）
- Vue Router（路由管理）
- Axios（网络请求）

## 🧩项目安装
1. 克隆项目后进入frontend文件
2. 安装依赖：`npm install`
3. 启动开发服务器：`npm run dev`
4. 打开backend下的.exe文件
5. 登录用户名为admin，密码为123456

## 📄注意事项
- 本项目是前端学习的练手项目，存在诸多问题望指正交流
- 开发环境依赖后端接口代理，配置见 `vite.config.js`
- 部分冗余代码已在文档中标注，可根据需要清理以优化项目结构
